# 車庫入れシミュレーター (Car Parking Simulator)

ブラウザ上で動作する、リアルな挙動の車庫入れシミュレーションアプリです。
車の運動学（アッカーマン・ステアリング・ジオメトリ）に基づいた動きを再現しており、前輪と後輪の軌跡を表示することで、車庫入れ時の**「内輪差」や「外輪差」を視覚的に理解**することができます。

## 🌟 特徴 (Features)

* **3つの駐車パターン**: 「バック駐車（直角）」「縦列駐車（左側）」「縦列駐車（右側）」の練習が可能です。

* **リアルなハンドル操作**: ドラッグ操作でぐるぐると回せるハンドルUIを実装。実際の運転に近い感覚で操作できます。

* **軌跡の可視化**: 前輪（オレンジ）と後輪（ブルー）が通った跡が線で描画され、タイヤの動きの違いが一目でわかります。

* **衝突検知**: ダミー車両や壁に接触すると画面が赤く光り、「衝突！」と警告が出ます。

* **依存関係ゼロ**: 外部ライブラリ（ReactやThree.jsなど）を使用せず、ピュアなHTML5 CanvasとJavaScriptのみで構築されているため、動作が非常に軽量です。

## 🎮 操作方法 (Controls)

### マウス / タッチ操作

1. **シフト選択**: 画面下の「前進(D)」「停止(P)」「バック(R)」ボタンをクリックして進行方向を決めます。

2. **ハンドル操作**: ハンドルのアイコンをマウスでドラッグ（スマホの場合はスワイプ）して回します。

3. 車が動いている間、タイヤの軌跡が描画されます。

### キーボード操作（PC推奨）

* **[ ↑ ] (上矢印)**: 前進 (Dギア)

* **[ ↓ ] (下矢印)**: バック (Rギア)

* **[ Space ] (スペース)**: 停止 (Pギア)

* **[ ← ] / [ → ] (左右矢印)**: ハンドルを左右に切る

## 🛠 技術スタック (Technologies Used)

* HTML5 (`<canvas>` API)

* CSS3 (CSS Variables, Flexbox, Grid)

* Vanilla JavaScript (ES6+)

## 🚀 実行方法 (How to Run)

このアプリケーションは単一のHTMLファイルで構成されているため、特別なサーバーやビルド環境は必要ありません。

1. このリポジトリをクローン、または `index.html` をダウンロードします。

2. `index.html` をお好みのWebブラウザ（Chrome, Firefox, Safari, Edgeなど）で開くだけで即座にプレイ可能です。

### 友達へのシェア方法

[Netlify Drop](https://app.netlify.com/drop) や GitHub Pages を利用して `index.html` をホスティングすることで、スマホやPC問わずURLひとつで簡単に共有できます。
